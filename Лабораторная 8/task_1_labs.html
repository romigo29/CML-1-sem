    CSS (Cascading Style Sheets) — каскадные таблицы стилей, которые предназначены для придания HTML-документам внешний вид.
    Внешнее подключение является наиболее удобным способом использования стилей и сокращает время обновления web- страниц. Внешние таблицы стили хранятся в отдельном файле, который может быть использован для любых веб-страниц и для подключения используется тег link внутри тега head.
    Цвет текста задается с помощью свойства color. Можно задавать разными способами: по шестнадцатеричному значению, по названию, в формате RGB, RGBA, HSL, HSLA. Свойство text-decoration добавляет оформление текста в виде его подчеркивания, перечеркивания, линии над текстом и мигания.
